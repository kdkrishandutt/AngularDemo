Steps:

    1.Create new Parent Component called SeversWithParentchild
    2.Create Two new child Component called ServerHeader and SeverDetails
    3.Add Child Component ServerHeader and SeverDetails with ngfor in parent Component SeversWithParentchild.
    4.Add Custom Property serverElements in Parent Compenent SeversWithParentchild.Ex. serverElements = [];
    5.Add Custom Property element in child Component SeverDetails.
    6.Add Input decorator for Property element in child Component SeverDetails and define alias Name for this decorator.
    7.Use this Input decorator in Parent Component to assign value to property. ex: [ServElement]="SERVERelement"
    8.Add Custom Event for Server and BulePrint in Parent Compenent Ex. (ServerCreated)="onServerAdded($event)" (BluePrintCreated) ="onAddBlueprint($event)"
    9.Add method for Server and BulePrint in Parent Component Ts file. Ex. onServerAdded, onAddBlueprint
    10.Add Custom Property in another child component ServerHeader. Ex. ServerCreated, BluePrintCreated
    11.Initialize Custom Property with EventEmitter child component ServerHeader. Ex. ServerCreated=new EventEmitter
    12.Add Output decorator in child component ServerHeader. Ex. ServerCreated, BluePrintCreated
    13.Emit both Custom property in method of child component ServerHeader Ts file. Ex. onAddServer, onAddBlueprint

    Custom Property:

    We can define our own property and decorate this with an decorator i.e input or output and use this property outside the component in another component.

    Input decorator:

    If you want to pass data from the parent component to the child component, then you need to use two things: @Input and property binding.

    Output decorator:

    If you want to pass data from the child component to the Parent component, then you need to use two things: @Output and new EventEmitter.

    EventEmitter:

    EventEmitter is a class in angular framework. It has emit() method that emits custom events. We can use EventEmitter in custom event binding
