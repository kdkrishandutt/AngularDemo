Steps:

    1.add ngx-translate your Angular application:

    npm install @ngx-translate/core @ngx-translate/http-loader rxjs --save
    2.Now you have to init the translation TranslateModule in your app.module.ts:
    import (TranslateLoader, TranslateModule) from '@ngx-translate/core';

    import (TranslateHttpLoader) from '@ngx-translate/http-loader';

    import (HttpClient, HttpClientModule) from '@angular/common/http';

    {{ TranslateModule }}
    3.Add below code in your Component:
    import (TranslateService) from '@ngx-translate/core';

    constructor(private translate: TranslateService) [ translate.setDefaultLang('en');]
    The next step is to set the default language of your application using translate.setDefaultLang('en'). In a real app you can of course load language from the user's settings.
    4.Each language is stored in a separate .json file. Let's create the JSON file for the English translation: assets/i18n/en.json.
    5.You have two choices when it comes to adding translations:
    translation pipe — (('id' | translate))
    translation directive — <element [translate]="'id'"></element>
    Both options lead to the same result — it's just a matter of personal tast which one you want to use.
    6.For more referance refer click here:Link
    7.Use this Input decorator in Parent Component to assign value to property. ex: [ServElement]="SERVERelement"
